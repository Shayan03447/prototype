# TRY N STYLE - AR Smart Mirror

AI-powered smart mirror prototype for virtual clothing try-on using augmented reality and body tracking.

## Features
- Real-time camera feed
- AI body tracking
- Virtual outfit overlay
- Size-fit indication
- AR smart mirror experience

## Tech Stack
- Python
- FastAPI
- OpenCV
- MediaPipe
- React

## Project Status
Prototype / MVP Development

## Goal
Reduce trial room dependency and improve retail shopping experience using AI + AR.
